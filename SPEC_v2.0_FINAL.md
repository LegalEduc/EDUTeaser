# 리걸크루 변호사 실전 압축 부트캠프 — 강사섭외 시스템 스펙

> 버전: v2.0 (최종) | 작성일: 2026-03-13
> v1.5 → v2.0: 프로토타입 확정 내용 반영 (멘토 명칭, 모달 폼, 강사료 제거, 디자인 확정, 문의처 확정)

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 리걸크루 변호사 실전 압축 부트캠프 — The Rookie Camp (강사섭외 시스템) |
| 브랜드명 | LegalCrew Academy |
| 목적 | 교육 프로그램 강사(리걸 커리어 멘토) 섭외 및 동의서 관리 자동화 |
| 플랫폼 | Netlify (서버리스) |
| 스택 | Next.js 15 (App Router) + Netlify Functions + Netlify DB (PostgreSQL/Neon) |
| ORM | Drizzle ORM |
| 이메일 | Resend (무료 100통/일) |
| PDF 생성 | pdf-lib (서버리스 함수 내) |
| 파일 저장 | Netlify Blobs (PDF 저장용) |
| 스타일링 | Tailwind CSS |
| 배포 | Netlify (Git 연동 자동배포, GitHub) |

## 2. 사용자 역할

| 역할 | 설명 | 접근 방식 |
|------|------|-----------|
| 어드민 | 프로그램 운영자 | 어드민 페이지 (비밀번호 보호) |
| 강사(멘토) | 섭외 대상 강사 | 티저 페이지(공개) + 동의서 링크(개별 이메일) |

---

## 3. 전체 흐름

```
[1단계: 참여 신청]
강사 → 티저 페이지 확인 → "멘토 신청" 버튼 클릭 → 모달 팝업 → 기본정보 입력 → 제출
                                                    ↓
[2단계: 어드민 세팅]
어드민 → 신청 접수 확인 → 강의주제, 강사료 금액, 특약사항 입력 → 동의서 링크 이메일 발송
                                                    ↓
[3단계: 동의서 서명]
강사 → 이메일 링크 클릭 → 동의서 내용 확인 (금액 등 포함) → 체크박스 + 서명 → 제출 → PDF 다운
```

### 4.2 참여 신청 모달 (팝업)

- **안내 문구**: "검토 후 강의 조건이 포함된 동의서 링크를 이메일로 보내드립니다."

### 4.3 동의서 서명 페이지 (개별 링크)

- **경로**: `/consent/{고유토큰}`
- **접근**: 어드민이 세팅 후 발송한 이메일 링크로만 접근

#### 동의서에 표시되는 내용 (어드민이 세팅한 값)

| 항목 | 설명 |
|------|------|
| 강사명 | 신청 시 입력한 이름 (자동) |
| 강의주제/과목명 | 어드민이 입력 |
| 강사료 금액 | 어드민이 입력 (원 단위) |
| 기타 특약사항 | 어드민이 입력 (자유 텍스트, 선택) |

#### 동의 항목 (체크박스)

| 동의항목 | 필수 |
|----------|------|
| 위 강의주제 및 내용에 대한 확인 | ✅ |
| 강사료 금액 및 지급 조건 동의 | ✅ |
| 개인정보 수집·이용 동의 | ✅ |
| 고유식별정보(주민등록번호) 수집·이용 동의 | ✅ |
| 초상권 및 강의 촬영·활용 동의 | 선택 |

#### 서명 + 제출 후

- 모든 필수 체크 + 본인 이름 입력 (신청 시 이름과 일치 검증)
- PDF 자동 생성 → 강사 즉시 다운로드 가능
- 어드민에게 서명 완료 알림 이메일

### 4.4 어드민 페이지 (비밀번호 보호)

- **경로**: `/admin`
- **접근**: 비밀번호 인증 (간단한 패스워드 방식, MVP 기준)

#### 대시보드
- 전체 현황: 신청 접수 / 동의서 발송(대기중) / 동의 완료

#### 강사 관리
- 신청 접수 목록 → 강사 기본정보 확인
- 동의서 세팅 (강의주제/강사료 금액/특약사항) → 링크 발송
- 동의 완료 목록 → PDF 다운로드 + 상세 조회

#### 공지 관리
- 공지 작성 (제목 + 본문) → 대상 선택 (전체/동의완료만) → 이메일 발송

## 7. PDF 동의서 생성

### 포함 내용

- 프로그램명 + 문서 제목 (강의 동의서)
- 강사 정보: 이름, 휴대폰, 이메일
- 강의 조건: 주제/과목명, 강사료 금액
- 특약사항 (있는 경우)
- 동의 항목별 체크 여부
- 서명 이름 + 서명 일시
- 계좌 정보: 은행명, 계좌번호, 예금주
- 주차 정보: 필요 여부 + 차량번호
- 하단: 고유 문서번호, 생성일시

### 기술

- `pdf-lib`로 서버리스 함수 내 생성 (Netlify Functions 10초 제한 내)
- Netlify Blobs에 저장

## 9. 보안 고려사항

| 항목 | 방안 |
|------|------|
| 주민등록번호 | AES-256 암호화, 어드민 조회 시 마스킹, 원천징수 시에만 복호화 |

| 토큰 만료 | 서명 후 비활성화 (PDF 다운만 가능) |
| 신청 폼 | rate limiting (IP/이메일 중복 방지) |
| 법적 근거 | 개인정보보호법 제24조 — 고유식별정보 별도 동의 |

## 10. 시스템 범위 외 (수동)

| 항목 | 처리 방식 |
|------|-----------|
| 강의자료 수집 | 이메일로 요청/수집 → 구글 드라이브 저장 |
| 강사료 지급 | 어드민이 계좌정보 확인 후 수동 이체 |
